Designing a MES Smart Tooling Feature for Wabtec Corporation

Date: May 2024 - July 2024

Team: MES and Smart MFG

Role: UI/UX Intern

Design Tools: Figma

Terms and Definitions:

  • Manufacturing Execution System: Software that optimizes the manufacturing process by monitoring, tracking, documenting, and controlling the entire production lifecycle.
  • Smart Tooling: a feature of the MES which displays the data that is collected from torque tools, gauges, ovens, and various other smart tools

Project description: Wabtec's current Manufacturing Execution System contains a feature called Smart Tools. Currently, the feature involves a series of steps which involved the shopfloor operators having to click on images and see various pop screens, leading to screen clutter.

My Role: My goal was to streamline the these steps within the Smart Tools feature so that screen clutter is reduced and navigation is made easy for the operators. Within the new interface, I organized the navigation where each field is enabled only when the previous input is answered and hid certain elements so that they only appeared when a pause and play button is pressed, and I also displayed the statuses on the same screen so that the need for a new pop up is eliminated.
