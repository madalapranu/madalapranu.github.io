Designing a Packing Slip App for Zebra Scanners at Wabtec Corporation

Date: May 2024 - July 2024

Team: Edison - Allison Flat, Hayden Fisher, Rachel Mathew, Cassidy Como

Role: UX Engineer and Project Manager

Design Tools: Figma

Development Tools: XML, Java, Postman, Android Studio

Project Description: The Wabtec site in Piossasco, Italy uses an old Enterprise Resource Planning system for processing receipt transactions for shipments. ERP, which stands for Enterprise resource planning, refers to a software that is used daily to manage day-to-day tasks across the company. Which means up to date ERP systems are extremely crucial in managing thousands of businesses among all industries. Wabtec's mission is to migrate this site over to Oracle in order to increase efficiency and convenience when processing these shipments. To assist with the migration to Oracle the site will be receiving new processing devices called Zebra TC58s. The new Zebra will be able to scan a QR code or ASN barcode that contains PO lines related to a certain shipment. With that, our team's task is to create an Android application to work in unison with the scanner. Once QR code is read, the operators will be able to see specific details relating to the shipment immediately. Then they will be able to accept or reject any lines displayed. Only once accepted, the information will then be sent to Oracle.

My Role: As the team's UI/UX designer, my goal was to provide the site's shopfloor operators the scanning capability, display the requesting packing slip information and receipt details, and intuitive screen navigation without cluttering up the Zebra Scanner's screen space.
